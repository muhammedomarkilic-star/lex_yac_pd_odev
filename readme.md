### CENG 218 Programlama Dilleri Öğrenci Proje Dosyaları

| Dosya Adı | Açıklama |
| :--- | :--- |
| ast.c | AST oluşturma ve değerlendirme fonksiyonunu barındıran dosya (bilmoodle'a yüklenecek) |
| lexer.l | Lexer dosyası (bilmoodle'a yüklenecek) |
| parser.y | Parser dosyası (bilmoodle'a yüklenecek) |
| data_loader.c | Verileri yükleme fonksiyonunu barındıran dosya |
| data_loader.h | Verileri yükleme fonksiyonu başlık dosyası |
| generate_inventory.py | Örnek envanter oluşturmak için kullanılan python kod dosyası. Kullanımı: `python generate_inventory.py <envanter_boyutu> <rastgele_tohum_degeri>` |
| inventory.csv | Envanterdeki pokemonların ozelliklerini barındıran dosya (generate_inventory.py ile yeniden oluşturulabilir) |
| main.c | Verileri yükleyen ve envanterde bir adet sorgu gerçekleştiren main fonksiyonunu barındıran dosya |
| moves.csv | Pokemonların hareketlerini barındıran dosya (generate_inventory.py için gerekli) |
| pogo_types.h | Gerekli yapıları ve öğrencilerin yazması gereken fonksiyonların prototiplerini barındıran dosya |
| pokedex.csv | Pokemonların ozelliklerini barındıran dosya (generate_inventory.py için gerekli) |
| pokemon_moves.csv | Pokemonların hareketlerini barındıran dosya (generate_inventory.py için gerekli) |
| type_chart.csv | Pokemonların tür etkinliklerini barındıran dosya |
| types.csv | Pokemonların türlerini barındıran dosya (generate_inventory.py için gerekli) |
